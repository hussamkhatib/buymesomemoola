## Installation  

Clone the project
```bash
git clone https://github.com/hussamkhatib/buymesomemoola.git  
```
Go to the project directory
```bash
cd buymesomemoola
```
Install dependencies
```bash
npm install
```
Start the server
```bash
npm run dev
```
    
## Tech Stack

- Next.js
- MongoDB
- DaisyUI
